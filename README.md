# SOC Analyst Home Lab — Practical Detection Projects

Репозиторий с серией мини-проектов, созданных для получения практического опыта SOC-аналитика.  
Каждый проект — это отдельный этап развития лаборатории: от установки SIEM до настройки расширенного мониторинга и анализа событий.

Цель лаборатории — сформировать практические навыки:
- развертывания SIEM;
- подключения и конфигурации агентов;
- мониторинга Windows-систем;
- анализа событий (FIM, Sysmon);
- создания отчётов и документации;
- выявления подозрительной активности.

---

# Структура репозитория

- /wazuh-base-setup → Базовая установка Wazuh Manager + Windows Agent
- /extended-fim → Расширенный File Integrity Monitoring

---

# Проект 1 — Wazuh Base Setup  
**Папка:** `/wazuh-base-setup`

Содержит базовое развертывание Wazuh:

- установка Wazuh Manager (Ubuntu)  
- установка Windows-агента  
- базовый пример File Integrity Monitoring  
- разбор первых событий  

Документация:
- Installation steps  
- Base FIM use case  
- Event analysis  

---

# Проект 2 — Extended File Integrity Monitoring  
**Папка:** `/extended-fim`

Расширенная конфигурация FIM для отслеживания появления исполняемых файлов и скриптов:

- `AppData\Local`
- `ProgramData`
- `AppData\Local\Temp`

Мониторинг файловых типов:
- `.exe`
- `.dll`
- `.ps1`
- `.bat`

В проекте:
- собственная FIM-конфигурация  
- тестовые сценарии  
- результаты и скриншоты  
- финальный отчёт
