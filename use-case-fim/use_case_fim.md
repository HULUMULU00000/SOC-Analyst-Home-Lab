# File Integrity Monitoring — Use Case

## Цель
Проверить, как Wazuh реагирует на создание, изменение и удаление файлов на Windows агенте.

## Шаги
1. Подключил Windows Agent
2. Настроил мониторинг директории
3. Создал файл → проверил события в Dashboard
4. Удалил файл → проверил события
5. Сравнил типы алертов
